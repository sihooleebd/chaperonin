What is Chaperonin?

A canvas for
protein pipelines

Chaperonin is a typed, visual workflow engine for computational biology. Drag modules onto a node canvas, wire their inputs and outputs together, and hit Run. The frontend serializes the graph and sends it to a Python backend over WebSocket, which executes each tool in order and streams live logs back.

The type system prevents bad connections — a Structure.PDB output won't connect to a Sequence.FASTA input. Every tool's resource requirements (GPU, memory) are declared upfront.

01

Drag modules

Pull any tool from the palette onto the canvas. Each module exposes typed input and output handles. Tools are grouped by category: design, predict, refine, render, convert.

02

Wire connections

Drag from an output handle to a compatible input handle. Chaperonin enforces type compatibility — it won't let you connect a structure file to a sequence consumer. Unconnected required inputs block Run.

03

Run

Hit Run. Chaperonin serializes the graph to JSON, sends it to the Python orchestrator via WebSocket, and streams node.running, node.progress, and node.log events back in real time.
